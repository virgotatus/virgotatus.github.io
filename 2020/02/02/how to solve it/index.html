读书笔记《how to solve it》

Yilong 发表于 2020-02-02

解题伪代码

  1. 理解题目
  2. 制定方案（寻找思路）
  3. 执行方案
  4. loop 2，3
  5. 回顾

详细算法

一. 理解题目

问自己问题：

  1. 未知量是什么？
  2. 已知数据？
  3. 条件？
  4. 满足条件的方法？
  5. 条件是否足以确定未知量？（秩）—可解性
  6. 如何形式化表示这些量？如何解释这些概念与定义？
  7. 可解性问题。找出三元素 条件（已知）——[映射（假设）（how to solve）]——结论(未知Z)

二. 寻找思路 —解决

问自己问题：

  1. 找出以前的类似题目或定理——利用它
    1. 要得出他需要什么定理
    2. 已知未知量相似的题目
  2. 重新叙述
    1. 改变未知量
    2. 改变已知量
    3. 都改变？
    4. 普遍化
    5. 特殊化
    6. 类似的
  3. 分解与重组
    1. 满足部分条件？
    2. 你用到所有东西了吗？
    3. 如何去找出中间的一部使条件和结论更接近？有什么辅助东西可以用？

探索法小词典

三. 执行方案

  1. 检查每一步是否正确
  2. 直觉与形式
  3. 区分步骤先后与大小

四. 回顾

  1. 简化每个步骤
  2. 你能以不同的方式推导出，来验证这个结果吗？
  3. 你能一眼就看出吗？or 你能一步步形式化推导出吗？ ——直观化与形式化
  4. 你能在别的地方使用这道题目吗？ 1. 结论 2. 方法
  5. 请你自己再造一道题目/

具体记录：

  1. 也许值得收集那些在帮助学生时经常会提到的问题与提示，并加以收集 ——聚类教育
  2. 好问题与坏问题：具有启发性的问题

探索法小词典：

  1. 类比

    1. 相近问题
    2. 用结果？用方法？
    3. 特殊化与普通化，找规律，猜测
    4. 类比：
    • 两个系统之间的对象存在某种一一对应（同构）——精确类比
    • 两个数学对象联系十分紧密，且规则都差不多——点-线-面-体
    • 一对多的对应——（欠构）另一类精确对比

  2. 辅助元素

    1. 为了应用某一到定理（以前的某道题）我们需要引入一个东西
    2. 怎么引入，引入一个特别的东西
    3. 理解动机和目的
    4. 辅助线、辅助未知量
    5. 有风险;判断：看上去更容易、更有启发性、美学魅力
    6. 观察未知量，变化题目
    7. 等价辅助题目（三角相等与三边相等）——可逆约减（双向约减）
    8. 单向约减——特殊化普遍化
